# ATM Simulator System 

A desktop-based ATM Simulator built using Java Swing and MySQL.

# Features
- User Authentication (PIN based)
- Deposit Money
- Withdraw Money
- Fast Cash
- Balance Enquiry
- Mini Statement
- PIN Change

#  Technologies Used
- Java (Swing, AWT)
- MySQL Database
- JDBC Connectivity

# How to Run
1. Install Java (JDK 8+)
2. Setup MySQL Database
3. Import database tables
4. Run Login.java
